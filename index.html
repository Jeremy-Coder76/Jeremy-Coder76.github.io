    About This Project¶

    "First, solve the problem. Then, write the code." — John Johnson

    This is where I document my transformation from traditional network engineering to the world of automation, DevOps, and modern infrastructure practices.

    Core Modules¶

    import journey¶

    My path from pre-sales engineering through network consulting to automation mastery. Real stories, real challenges, real growth.

    import technical¶

    Code snippets, automation scripts, and infrastructure patterns. Python, Ansible, APIs, and everything in between.

    import reflections¶

    Thoughts on career pivots, continuous learning, and the mindset shifts required to thrive in tech.

    Quick Start¶

    from jleclech_blog import Blog

blog = Blog()

blog.explore_journey()
blog.view_technical()
blog.about_author()

if __name__ == "__main__":
    blog.run()

    Endpoint Method Description
    /journey GET Access career evolution stories
    /technical GET Retrieve code & tutorials
    /about GET Author information
    /latest GET Recent updates
